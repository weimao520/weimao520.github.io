idea

检出分支： 右下角checkout 你要检出的分支 切换分支一样

合并分支： 如我在dev 分支 写的代码想合并到master分支； 我先在dev 分支先提交 push 在切换到master分支

然后 去右下角 点击 dev 会看到merge into curent 这样会把 dev提交的代码合并到master分支，在（master）push

Git：命令

##  查看本地分支
git branch -a(所有分支，远程/本地  不加 显示本地分支)
## 切换分支
git checkout  -b  分支名

## 合并分支

git merge  dev  ##  把dev 分支合并到当前分支  提前是dev分支有提交代码   
## 这样当前分支就有 dev分支的代码 当前分支在push  提交下几可以
git 更换提交地址
##  先删除原来的地址
git remote rm origin

##  更换你要提交的地址
git remote add origin XXXX地址

如果账号密码有改动

git config --system  --unset credential.helper

##  如果用了第一个命令 还不能解决问题那么 用这个命令：
git config –global http.emptyAuth true
