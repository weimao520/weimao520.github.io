SpringCloud 配置网关的两种方式：

一种就是上面的配置的Yml 配置

还有一种就是写配置类

@Configuration
public class GatewayConfig {
    
    
    @Bean
    public RouteLocator routes(RouteLocatorBuilder builder){
        RouteLocatorBuilder.Builder build = builder.routes();
        //  访问 localhost：http://news.baidu.com/guonei" 就可以直接跳到该网站
        build.route( r -> r.path("/guonei"). uri("http://news.baidu.com/"));
        return build.build();
    }
}

断言的介绍及使用

SpringcloudCateway将路由匹配作为Spring WebFlux HandlerMapping基础框架的一部分。

SpringCloudCateway包括许多内置的RouterPredicate工厂，所有这些Predicate都与HTTP请求不同属性相匹配。多个RouterPredicate工厂可以进行组合。

Spring Cloud Gateway创建Route对象时，使用RoutePredicateFactory创建Predicate对象，Predicate对象可以赋值给Route。Spring Cloud Gateway包含许多内置的Route Predicate Factories。

所有这些谓词都匹配HTTP请求的不同属性。多种谓词工厂可以组合，并通过逻辑and..

使用：


spring:
  application:
    name: cloud-gateway
  cloud:
    gateway:
      discovery:
        locator:
          enabled: true  #开启从注册中心动态创建路由功能，利用微服务名进行路由
      routes:
        - id: payment_routh #路由的ID，没有固定规则但要求唯一，建议配合服务名
#          uri: http://localhost:8001   #匹配后提供服务的路由地址
          uri: lb://CLOUD-PAYMENT-SERVICE   #利用微服务名进行动态路由
          predicates:
            - Path=/payment/**   #断言,路径相匹配的进行路由
            ## 2020-03-08。。 之后路由才起作用
            - After=2020-03-08T10:59:34.102+08:00[Asia/Shanghai]
            之后路由之前起作用
            - Before=2020-03-08T10:59:34.102+08:00[Asia/Shanghai]
            ###  两个时间之间
             - Between=2020-03-08T10:59:34.102+08:00[Asia/Shanghai] ,  2020-03-08T10:59:34.102+08:00[Asia/Shanghai]
             ##  设置请求方式
            - Method=Get
            ##  # 需要两个参数，一个Cookie name ,一个正则表达式
#            - Cookie=username,WeiMao
			- Header=X-Request-Id, \d+   #请求头中要有X-Request-Id属性并且值为整数的正则表达式

PS:*Predicate 就是实现了一组匹配规则，让请求过来找到对应的Router进行处理 *

过滤器（Filter）

@Component
@Slf4j
public class MyLogGateWayFilter implements GlobalFilter,Ordered {
    @Override
    public Mono<Void> filter(ServerWebExchange exchange, GatewayFilterChain chain) {

        log.info("*********come in MyLogGateWayFilter: "+new Date());
        String uname = exchange.getRequest().getQueryParams().getFirst("username");
        if(StringUtils.isEmpty(username)){
            log.info("*****用户名为Null 非法用户,(┬＿┬)");
            exchange.getResponse().setStatusCode(HttpStatus.NOT_ACCEPTABLE);//给人家一个回应
            return exchange.getResponse().setComplete();
        }
        return chain.filter(exchange);
    }

    
    @Override
    public int getOrder() {
        return 0;
    }
}
