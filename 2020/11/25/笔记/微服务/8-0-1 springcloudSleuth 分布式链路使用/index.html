使用：

​ Springcloud从F 版开始就不需要手动构建Zipkin server 了，只需要从官网 下载jar就好。

然后在运行其jar包，访问 http://localhost:9411 就可以访问其web页面

​

服务提供者：

pom

<!--包含了sleuth+zipkin-->
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-starter-zipkin</artifactId>
        </dependency>

YML

server:
	port: 8081
spring:
  application:
    name: cloud-provide-service
  zipkin:
    base-url: http://localhost:9411
  sleuth:
    sampler:
    probability: 1 #采样率值介于0~1之间，1表示全部采集

启动类

@SpringBootApplication
public classs Application{
    public static void main(String [] args){
        SpringApplication.run(Application.class,args);
    } 
}

测试Controller

@RestController
public class ProvideController {
    
    @GetMapping("/provide/zipkin")
    public String testZipkin(){
        return "hello，zipkin";
    }
}

服务消费方：

pom

<!--包含了sleuth+zipkin-->
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-starter-zipkin</artifactId>
        </dependency>

YML

server:
	port: 80
spring:
  application:
    name: cloud-test-service
  zipkin:
    base-url: http://localhost:9411
  sleuth:
    sampler:
    probability: 1 #采样率值介于0~1之间，1表示全部采集

启动类

@SpringBootApplication
public classs Application{
    public static void main(String [] args){
        SpringApplication.run(Application.class,args);
    } 
}

测试Controller 访问提供方

@RestController
public class TestController{
    @Autowired
    private RestTemplate restTemplate;
    
    @GetMapping("test/zipkin")
    public String testZipkin(){
        return this.restTemplate.getForObject("http://localhost:8081/provide/zipkin", String.class);
    }
}

访问 http://localhost/test/zipkin 测试。成功的话 页面会打印 hello，zipkin

再去访问 http://localhost:9411 访问zipkin页面 选择依赖

会发现 之前发送的请求 的执行链路包括时间

cloud-test-service ——–> cloud-provide-service
